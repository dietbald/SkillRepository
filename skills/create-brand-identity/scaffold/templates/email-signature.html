How to install this signature

Gmail: Settings → See all settings → General → Signature → Create new → paste the rendered preview below (use Ctrl/Cmd+A to select all in the preview frame, then Ctrl/Cmd+C). Save changes.

Outlook (desktop): File → Options → Mail → Signatures → New → paste rendered preview.

Apple Mail: Mail → Preferences → Signatures → "+" → paste rendered preview. Uncheck "Always match my default message font."
