# Newsfeed

## What this area covers

The system-wide announcement stream Halingo publishes to all signed-in users — release notes, regulatory news, planned-maintenance notices, and tips relevant to Belgian logopedist practices.

## Features in this area

- Dashboard widget — the latest items appear in a tile on the main dashboard so users see them on every login.

## Key product behaviors

- The newsfeed is **read-only** for end users. There is no in-app authoring UI; content is curated by Halingo and seeded into the product centrally.
- Each item carries a Dutch (NL) and French (FR) version; the user sees the version that matches their UI language preference.
- Items are time-ordered; the dashboard widget surfaces the most recent few, with a link to the full feed.
- Items are not personalised — every user sees the same announcements.

## Cross-references

- Main Dashboard — the newsfeed widget is one of the dashboard tiles.
- Identity — the user's UI language preference determines whether the NL or FR version of each item is shown.
