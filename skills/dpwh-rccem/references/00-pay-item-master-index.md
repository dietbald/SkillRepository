## Notes on the Master Index

1. **Doc Page** refers to the page number within the RCCEM manual (not the PDF page/image number).
2. Page numbers in the index match the DUPA sheet location where full cost breakdowns appear.
3. The summary table in the RCCEM appears on manual pages 5–13 (corresponding to image pages 011–019).
4. Items with duplicate item numbers (e.g., multiple 311(1)b.x entries) represent thickness/capacity variants that each have their own DUPA sheet.
5. Part designations: **C** = Earthworks, **D** = Subbase & Base Course, **E** = Surface Course, **F** = Drainage & Slope Protection Structures, **H** = Miscellaneous Structures.
6. Bio-Engineering Solutions items (6201–6203 and 621-series) cover slope protection using natural/organic materials.
